Task Execution & Challenges

Today, I focused on improving how APIs are structured — making them more modular, scalable, and secure. I explored different strategies to handle real-time interactions such as using long-polling, socket communication, and basic event triggers. Additionally, I learned about handling cookies, headers, and security middleware like CORS and Helmet in Express.

Alongside backend learning, I also spent a good amount of time refining the UI/UX of my personal portfolio site. I focused on improving layout responsiveness and interactive elements using Tailwind CSS and React.

Challenges I faced:
  • 🌀 Managing state and feedback in real-time API responses required careful event flow control.
  • 🔐 Handling cookies securely without exposing user data during API testing was tricky initially.
  • 📱 Ensuring mobile responsiveness and clean transitions in my portfolio layout took multiple tweaks.
