Dived into the world of Node.js — explored its core concepts, routing, authentication, and server handling.

Start of the Day (SOD)

SOD 6 July 2025

My main focus today was:

  • 🧠 Start learning Node.js backend fundamentals through this Node.js YouTube course.
  • 📝 Write and publish Day 12 daily blog.

Task Execution & Challenges

I began learning Node.js from scratch — covering the basics like modules, creating a server, and understanding request-response flow. I then moved into more hands-on areas like routing, CRUD operations, and middleware handling.

Later in the day, I explored advanced topics like authentication, working with cookies, and using Express for building APIs.

Challenges faced:

  • 🧱 Understanding how middleware works behind the scenes took a bit of time to click, especially with Express.
  • 🔐 Managing user sessions and cookies during authentication flow was tricky, especially without frontend context.
  • 📦 Keeping all dependencies and files structured properly required some refactoring midway through learning.
  • 🧭 Switching from frontend mindset to backend logic felt like shifting gears, but I’m starting to enjoy it!

End of the Day (EOD)

  • ⚙️ Some concepts like deployment and error handling still pending — I plan to complete them tomorrow.
  • ✍️ Finished writing and published today's blog entry.
