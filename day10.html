Dived into backend structure and API handling using Fetch and Axios.

Start of the Day (SOD)

SOD 4 July 2025

Goals for the day:

  • 🧠 Learn and understand the basic backend structure.
  • 🔁 Explore how to make GET, POST, PUT, and DELETE requests using both Fetch API and Axios.
  • ⚙️ Continue polishing my portfolio website.
  • 📝 Wrap up the day with this blog entry.

Task Execution & Challenges

I began by reviewing how a backend system handles HTTP requests — endpoints, routing, and data operations. Using tools like Postman helped me visualize the request-response flow. Then, I practiced making CRUD operations using both Fetch and Axios in the frontend.

Here are the challenges I encountered:

  • 🔗 Understanding the differences in Fetch vs Axios syntax for handling responses and errors.
  • 🧪 Struggled a bit with setting correct headers (especially for POST and PUT requests with JSON payloads).
  • 🔄 Managing state in the frontend while receiving data from APIs required debugging and console tracking.
  • 🎨 Continued updating and improving my portfolio alongside — tweaking UI elements and testing responsiveness.

Despite minor roadblocks, today was a meaningful step in connecting frontend to backend effectively.

End of the Day (EOD)

  • ✅ Practiced API integration using both Fetch and Axios — still have a few areas to explore tomorrow.

  • ✅ Wrote and published Day 10 blog.
