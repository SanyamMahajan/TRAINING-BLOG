Task Execution & Challenges

I started the day by focusing on completing the remaining videos of the Namaste JS course. It covered topics like closures, hoisting, the call stack, and lexical environments — which helped deepen my understanding of how JavaScript works internally.

Post that, I shifted attention to the training blog website. I redesigned and polished the layout, improved mobile responsiveness, added features like scroll progress, back-to-top button, and better dark mode support.

Lastly, I worked a bit more on the portfolio project in React. Since a lot of time was invested in JS learning and UI enhancements today, I’ll finish the portfolio tomorrow.

Challenges faced today:

  • ⚙️ Had to revisit some JS concepts like closures and scopes a couple of times to fully grasp them.
  • 🛠️ Integrating scroll-based features and progress tracking in the blog site UI needed careful JS handling.
  • 🧩 Managing CSS responsiveness across devices while keeping animations smooth took time and testing.
