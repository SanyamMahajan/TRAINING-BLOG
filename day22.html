Training Blog - Day 22

Dived into Redux Toolkit and solved some key DSA problems from LeetCode.

Start of the Day (SOD)

Date: 18 July 2025

Planned tasks for today:

  • 🧠 Learn about Redux Toolkit and how it simplifies state management.
  • 💻 Solve DSA problems from Striver’s A2Z Sheet on LeetCode.
  • 📝 Write and publish the daily blog.

Task Execution & Challenges

Started the day by diving into Redux Toolkit with this video tutorial. I learned how Redux Toolkit reduces boilerplate code compared to traditional Redux and makes state management easier in large-scale React apps.

Some concepts explored today:

  • 🔁 Creating slices using createSlice
  • 🧱 Configuring the store with configureStore
  • 📦 Managing global state and dispatching actions easily
  • ⚙️ Using Redux DevTools to monitor app state in real time

After that, I practiced three DSA problems to sharpen my problem-solving skills:

  • 1️⃣ Two Sum
  • 2️⃣ Best Time to Buy and Sell Stock
  • 3️⃣ Next Permutation

Challenges faced:

  • 🔧 Understanding how immer works under the hood in Redux Toolkit for immutability was tricky at first.
  • 🔍 Debugging logic in the “Next Permutation” problem required carefully tracking indices and swaps.
  • 🧠 Managing time between learning Redux and solving DSA was a bit tight today.

End of the Day (EOD)

  • ✅ Completed core concepts of Redux Toolkit and its implementation using slices and stores.
  • ✅ Solved 3 DSA questions and practiced edge case thinking.
  • ✅ Blog written and published successfully.

  • Solving DSA problems: 3 hours 30 minutes

Conclusion

Today was packed with valuable learning. I now feel more confident using Redux Toolkit in React projects, and solving DSA problems added that perfect logical workout. Hoping to implement Redux in my next app soon!
