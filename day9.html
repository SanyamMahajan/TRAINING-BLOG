Start of the Day (SOD)

SOD 2 July 2025

Today’s key focus:

  • 🎨 Explore and understand core concepts of SCSS.

Task Execution & Challenges

I began the day by diving into SCSS (Sassy CSS). I covered variables, nesting, mixins, functions, and partials with `@import`. SCSS offers powerful tools for cleaner, modular, and reusable styling — which makes larger projects more manageable.

After that, I moved on to further refining my portfolio website’s UI and structure. I added new sections, improved mobile responsiveness, and reviewed accessibility basics.

Challenges faced:

  • 📌 Understanding SCSS nesting and scope required careful structuring — deep nesting quickly became hard to maintain.
  • ⚙️ Integrating SCSS into Vite required configuration via `vite.config.js` and installing `sass` — was a good learning experience.
  • 🎯 While enhancing portfolio UI, I ran into layout issues across different screen sizes which needed custom tweaks beyond Tailwind defaults.

End of the Day (EOD)

  • ✅ Completed SCSS learning including variables, nesting, mixins, and partials.
  • ✅ Improved and enhanced the portfolio UI — added new animations and improved mobile responsiveness.
  • 📝 Wrote and published today’s daily blog entry.
