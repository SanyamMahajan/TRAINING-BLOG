Planned tasks for the day:

Task Execution & Challenges

Today was all about sharpening DSA skills. I picked up four interesting problems from Striver’s A2Z sheet and solved them on LeetCode:

  • 🔹 Pascal’s Triangle
  • 🔹 Rotate Matrix by 90 Degrees
  • 🔹 Print Matrix in Spiral Manner
  • 🔹 Count Subarray with Given Sum

While solving these problems, I focused not only on brute-force methods but also optimized my solutions using space and time-efficient logic. Understanding matrix traversal patterns and prefix sums for subarray problems was especially rewarding.

Challenges faced:

  • 🔁 Handling edge cases in spiral traversal took a few iterations to debug.
  • 🧮 Implementing the prefix sum and hash map logic for subarray sum was a bit tricky but satisfying once it worked.
  • 🧠 Remembering optimal matrix rotation logic without extra space required a clear understanding of in-place swaps.

Today’s deep dive into DSA problem solving was fruitful and engaging. It helped me not only practice algorithmic patterns but also reinforced my ability to write clean and optimal code. Looking forward to solving more complex problems in the upcoming days!
