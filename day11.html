Task Execution & Challenges

Today’s focus was all about making the backend-to-frontend connection seamless. I practiced calling APIs using both Fetch and Axios with proper headers, error handling, and real-time UI updates.

I also explored adding loading indicators and managing UI state based on response codes, which made the interface feel much more professional.

Challenges faced:

  • ⚠️ CORS issues while testing local APIs with fetch — had to dig into browser settings and headers.
  • 🔄 Difficulty maintaining consistent state updates when multiple requests were triggered together.
  • 🧩 Structuring API logic inside components without cluttering made me rethink some folder and file arrangements.
  • 🎯 Error handling patterns (try-catch vs .then-catch) gave me some confusion initially in mixed code.
