Task Execution & Challenges

Wrapped up the remaining HTML and CSS topics from the interview questions repo. Topics like CSS positioning, pseudo-classes, accessibility, and layout techniques were thoroughly reviewed.

Some challenges today:

  • 🔄 Switching context between HTML/CSS and JavaScript needed careful time management to retain focus.
